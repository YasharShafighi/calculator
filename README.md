# 🧮 Web Calculator

An online, interactive calculator built with **HTML, CSS, and JavaScript**. This project features an engineering-style interface and is optimized for use across various browsers.

## 🌐 Live Demo

No installation or download is required! Simply click the link below to start using the calculator:

👉 **[View and Use the Calculator](https://yasharshafighi.github.io/calculator/)**

## ✨ Features

- **Basic Operations**: Addition, subtraction, multiplication, and division
- **Advanced Functions**: Exponents, square root, percentages, and trigonometric functions (sin, cos, tan)
- **Memory Buttons**: Store and recall numbers (M+, M-, MR, MC)
- **Simulated UI**: A realistic, engineering-calculator-style interface
